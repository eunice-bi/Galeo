
Première version du programme

Ce programme en Python 3.6 génère un fichier Excel à partir du fichier "data.xlsx" qui doit être dans le même dossier que le fichier Pyhon. 
Ce fichhier Excel contient 14 feuilles : 
- le tableau sans colonnes vides
- les noms de toutes les colonnes
- les noms de toutes les colonnes vides
- les noms de toutes les colonnes sauf les vides
- les noms des colonnes écrites en français
- les noms des colonnes écrites en français + adresses des points écrits en français
- la moyenne des valeurs pour chaque colonne
- les min et max pour les colonnes aux noms écrits en français
- les min et max pour les colonnes non vides
- les moyennes, min et max pour chaque colonne non vide
- les valeurs uniques pour chaque colonne non vide
- les valeurs uniques pour les colonnes aux noms écrits en français
- les colonnes répétées à l'identique
- les modèles des colonnes répétées à l'identique
